    Keys:

  • drag nodes
  • click on element to select/deselect
  • click on field to cancel selection
  • drag field
  • scroll to zoom
  • double click to edit element
  • enter in edit mode to finish editing
  • escape in edit mode to cancel editing
  • n - new node
  • r - reverse arcs
  • c - connect nodes
  • delete, backspace - delete objects
  • escape - cancel selection
  • ctrl+Z - undo
  • ctrl+Y - redo
  • ctrl+S - save to JSON file
  • drag & drop JSON file - load from file
